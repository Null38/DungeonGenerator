# DungeonGenerator

[졸업 작품](https://github.com/Null38/DungeonDecktics)에 사용하기 위해 만들었다만, 시간 부족으로 인해 활용하지 못한 던전 제너레이터 입니다.

[Noel Berry](https://github.com/NoelFB)의 [절차적 생성 - 던전](https://noelberry.ca/posts/thedungeons/)을 기반으로 만들어져있습니다.

포트폴리오에 올리기 위해 뒤늦게 올리다만, 졸업 작품의 이전[커밋](https://github.com/Null38/DungeonDecktics/tree/8b6893eb3292dcfd340ae3108301e2af47f5cbc9)에서 당시 작업하였음을 확인 가능합니다.
